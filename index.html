# 💼 PortageOS - Dashboard Financier Une application web locale conçue pour piloter, budgétiser et optimiser ses finances personnelles. Pensée en priorité pour les consultants en **Portage Salarial**, elle s'adapte également parfaitement au statut de salarié classique. ## ✨ Fonctionnalités Principales * **🎛️ Simulateur de Rémunération :** Calcul précis du "Super Net" et du pouvoir d'achat total à partir du Chiffre d'Affaires HT (frais de portage, indemnités kilométriques, chèques culture, PEE, etc.). * **💰 Suivi Budgétaire :** Interface de budgétisation des charges fixes (loyer, énergie), des dépenses de vie, des remboursements de dettes et des investissements manuels (PEA, Crypto). * **📊 Bilan Analytique :** Tableau de bord global avec calcul automatique du Reste en Banque et du **Taux d'Épargne Réel**. * **📥 Exportation :** Génération et téléchargement de l'historique complet au format CSV pour un traitement sur Excel. ## 🛠️ Stack Technique * **Backend :** Python 3 (Framework Flask) * **Base de données :** SQLite3 * **Frontend :** HTML5, CSS3, Vanilla JavaScript (zéro dépendance externe) * **Hébergement cible :** Conteneur LXC Linux ## 🚀 Installation & Lancement L'application est conçue pour tourner en réseau local. 1. Cloner le dépôt sur le serveur cible : ```bash git clone [https://github.com/TonPseudo/Kpi_Finance_Personelle.git](https://github.com/TonPseudo/Kpi_Finance_Personelle.git) cd Kpi_Finance_Personelle Installer les dépendances requises : Bash pip install flask Lancer l'application : Bash python3 app.py L'interface sera alors accessible depuis un navigateur web via l'adresse IP du serveur sur le port 5000 (ex: http://192.168.1.100:5000).
